# Demo_PDFmerger_JS
Just a demo of creating a pdf merger application using bootstrap and vanilla javascript

## Instalation for Testing The App
```git clone https://github.com/ch3tanbug/Demo_PDFmerger_JS.git```

```cd Demo_PDFmerger_JS```

```npm ci```

```node server.js```
or
```nodemon server.js```
